# 更新日志

本项目的所有重要更改都将记录在此文件中。

本项目遵循 [语义化版本](https://semver.org/spec/v2.0.0.html)。

## v1.1.1
- 修复了修改课程后删除按钮失效的问题
- 修复了部分情况下无法从文件导入课程表的异常
- 优化了在大屏幕设备上的显示效果
- 在全部课程表页面进入多选模式时，未选中课程表时自动隐藏删除按钮
- 在全部课程表页面显示学期起止日期
- 在课程表及课程表时间设置页面新增提示

## v1.1.0

### ⚠️ 破坏性变更
- **Windows 配置文件路径变更**：
    - 安装版（exe、msi）：从 `C:\Users\<账户>\AppData\Local\Lyxot\TimeFlow` 迁移到 `C:\Users\<账户>\AppData\Roaming\TimeFlow`
    - 便携版（zip）：从 `C:\Users\<账户>\AppData\Local\Lyxot\TimeFlow` 迁移到可执行文件所在目录下

### 新增功能
- 支持从文件导入和导出课程表
- 支持长按拖动快速选择或取消选择上课周数
- 点击已选中的"单周/双周/全选"按钮可快速清空周数选择
- 在设置页面显示当前配置文件保存路径 (仅桌面端)

### 界面优化
- 统一各页面间距，优化视觉一致性
- 支持 Windows 系统强调色主题
- Windows 和 macOS 支持沉浸式标题栏
- 桌面端窗口图标随主题切换（浅色/深色）
- 使用最新的 Material Design 色彩规范（2025）

## v1.0.2
- 修复在 Windows 系统上无法加载配置文件的问题

## v1.0.1
- 优化日期选择器显示，修复组件过小的问题
- 修复在创建同一时段课程时，默认选择全部星期的异常
- 解决因频繁重组导致的高 CPU 占用问题（Android）
- 校验并限制学期起止日期，使其保持在合理范围内

## v1.0.0

- TimeFlow 初始版本发布。
- 核心功能:
    - 跨平台支持 (Android, iOS, 桌面端)。
    - 课程表与课程管理。
    - 支持浅色与深色模式。
