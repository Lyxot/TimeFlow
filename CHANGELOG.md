# 更新日志

本项目的所有重要更改都将记录在此文件中。

本项目遵循 [语义化版本](https://semver.org/spec/v2.0.0.html)。

## v1.3.1

### 新增功能

- 支持将课程表总览导出为图片，课程数据嵌入图片中，可从图片重新导入（需保持图片文件未被修改）
- 总览模式下始终显示周末列，无论是否设置为默认显示
- 网页端新增加载动画，优化首屏体验

### 修复问题

- 修复了总览模式下课程节数标签随行数增多逐渐偏移的对齐问题
- 修复了服务端环境变量配置不生效的问题

### 优化

- 加快网页端页面加载速度

### 实验性功能

- API 服务端 Docker 镜像：`lyxot/timeflow-server`，支持通过 `docker-compose` 一键部署

## v1.3.0

### 新增功能

- 新增课程表总览模式，以合并单元格展示所有周次的课程安排
- 课程表单元格行高可根据内容动态调整，并随窗口大小变化实时更新
- 为课程表名称、课程名称、教室、老师、备注添加字符长度限制

### 修复问题

- 修复了今日页面在当前周超出总周数时未显示假期周的问题

### 优化

- 增大课程表页面的底部边距，避免被浮动按钮遮挡
- 更新项目依赖
- 重构课程表页面，提升性能和可维护性

### 实验性功能

- 新增 Web 端支持（JS / WasmJS），支持通过浏览器访问课程表，使用浏览器 localStorage 持久化数据。[点此查看 Web 端演示](https://timeflow.hyli.xyz)（仅限演示，不保证可用性）
- API 服务端：完整的 REST API，包含 JWT (ES256) 认证、邮箱验证、课程表与课程 CRUD、PostgreSQL 数据库、Docker 部署支持 （仅后端，尚未集成至 APP）

## v1.2.0

### ⚠️ 破坏性变更

- 配置文件和导出的课程表现在以新的 `v2` 格式存储，并默认使用 gzip 压缩。此版本创建的数据文件与旧版本**不兼容**，无法被旧版应用读取 (但当前版本依旧可以读取旧的 v1 版本数据)。

### 新增功能

- 已删除的课程表现在会移入回收站，而不是被永久删除
- 为课程增加了新的备注字段
- 支持从课程表列表页导出指定课程表

### 修复问题

- 修复了删除课程后单元格分割线没有正确重绘的 Bug

### 优化

- 重构内部数据模型至 v2 版本，提升类型安全与运行效率
- 配置文件和导出数据默认启用 gzip 压缩
- 重构课程表 UI 的状态管理方式，提升界面响应性能
- 启用 iOS 端并行渲染
- 更新了应用的内部导航结构，使用类型安全的密封类来定义路由

## v1.1.1
- 修复了修改课程后删除按钮失效的问题
- 修复了部分情况下无法从文件导入课程表的异常
- 优化了在大屏幕设备上的显示效果
- 在全部课程表页面进入多选模式时，未选中课程表时自动隐藏删除按钮
- 在全部课程表页面显示学期起止日期
- 在课程表及课程表时间设置页面新增提示

## v1.1.0

### ⚠️ 破坏性变更
- **Windows 配置文件路径变更**：
    - 安装版（exe、msi）：从 `C:\Users\<账户>\AppData\Local\Lyxot\TimeFlow` 迁移到 `C:\Users\<账户>\AppData\Roaming\TimeFlow`
    - 便携版（zip）：从 `C:\Users\<账户>\AppData\Local\Lyxot\TimeFlow` 迁移到可执行文件所在目录下

### 新增功能
- 支持从文件导入和导出课程表
- 支持长按拖动快速选择或取消选择上课周数
- 点击已选中的"单周/双周/全选"按钮可快速清空周数选择
- 在设置页面显示当前配置文件保存路径 (仅桌面端)

### 界面优化
- 统一各页面间距，优化视觉一致性
- 支持 Windows 系统强调色主题
- Windows 和 macOS 支持沉浸式标题栏
- 桌面端窗口图标随主题切换（浅色/深色）
- 使用最新的 Material Design 色彩规范（2025）
